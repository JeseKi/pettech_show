当前节点

API 授权

企业微信朋友圈发布从应用授权和 access_token 开始，后续所有动作都通过官方接口链路执行。

输入

企业微信应用、客户联系权限、发布账号范围和内容资产。

处理

获取 access_token，并确认应用具备客户朋友圈相关接口权限。

效果

发布不依赖模拟点击，而是进入可追踪、可重试、可审计的 API 流程。
